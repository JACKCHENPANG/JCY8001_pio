# JCY8001 通信协议 (Modbus RTU)

**版本 v2.16** · 大唐恩智浦 DNB1101 电池阻抗单机 · 全部寄存器经实测验证

## 1. 物理层
| 项 | 值 |
|---|---|
| 协议 | Modbus RTU |
| 从机地址 | 1 |
| 波特率 | 115200 |
| 数据位/校验/停止 | 8 / 无 / 1 (8N1) |
| 接口 | USB (CP2102 虚拟串口) |
| 字节序 | 大端 (Modbus 标准); 64位阻抗值=4寄存器大端 |

## 2. 支持的功能码
| FC | 功能 | 用途 |
|---|---|---|
| 0x01 | 读线圈 | 读 ZM/均衡 启停状态 |
| 0x02 | 读离散输入 | 同上(只读) |
| 0x03 | 读保持寄存器 | 读所有数据/参数(主用) |
| 0x04 | 读输入寄存器 | 等同 FC03 |
| 0x05 | 写单线圈 | 触发 ZM/均衡/升级/保存校准 |
| 0x06 | 写单寄存器 | 设单个参数 |
| 0x0F | 写多线圈 | 批量线圈 |
| 0x10 | 写多寄存器 | 多参数 / OTA数据块 |

## 3. 数据寄存器 (FC03/04 读)
| 地址 | 名称 | 说明 / 换算 |
|---|---|---|
| 0x3300 | 温度 | 值/10 = ℃ |
| 0x3340 | 电压 | 值/10000 = V |
| 0x3380 | 状态 | 0x0001=测量中 0x0006=完成 0x0005=ADC错 |
| 0x3000~0x3003 | 阻抗实部 RE | 4寄存器=64位大端有符号, 值/100000 = μΩ |
| 0x3080~0x3083 | 阻抗虚部 IM | 同上(已取-Z_imag) |
| 0x3200 | VZM | 激励参考电压原始值 |

## 7b. 固件自主扫频 (EIS sweep, 单通道逐点)
芯片一次只测一个频率; 固件把"逐点切频+测量"循环搬到本机, 上位机只需下发频点表+触发, 之后边跑可边读 (轮询 0x3E41 已完成点数) 或等 0x3E40=2 完成后批量读。每点沿用单点 ZM 的平均/转换门/采样电阻逻辑。

| 地址 | 功能 |
|---|---|
| 0x4400~0x443F | (FC10写) 频点表, 每点1寄存器=16位 M/E 频率码 (同 0x4200 格式), 最多64点 |
| 0x43C0 | (FC06写) 频点数 N (1~64) |
| 线圈 0x00C0 | (FC05) 0xFF00=启动整段扫频, 0x0000=中止 |
| 0x3E40 | (FC03读) 扫频状态 0空闲/1跑中/2完成/3中止 |
| 0x3E41 | (FC03读) 已完成点数 |
| 0x3400+idx*4 | (FC03读) 第idx点 RE, 4寄存器=64位大端, /100000=μΩ |
| 0x3500+idx*4 | (FC03读) 第idx点 IM, 同上 |
| 0x3600+idx | (FC03读) 第idx点 VZM 原始 |
| 0x3640+idx | (FC03读) 第idx点实际所用频率码 (M/E) |

## 8. 典型流程
### 8.1 读温度电压
`FC03 0x3300 cnt=1` → 温度; `FC03 0x3340 cnt=1` → 电压

### 8.2 阻抗测量 (单点)
1. FC06 0x40C0 = 档位(2=1Ω)
2. FC06 0x4200 = 频率码(如0x0742=66.7Hz)
3. FC06 0x4340 = 1 (快速)
4. FC05 线圈 0x0000 = 0xFF00 (触发)
5. 轮询 FC03 0x3E2D, 触发后先等归0再等回1 (防读残值)
6. FC03 0x3000 cnt=4 读RE, 0x3080 cnt=4 读IM, /100000=μΩ
7. FC05 线圈 0x0000 = 0 (停, 恢复电压测量)

### 8.3 全谱扫描 (固件自主扫频, 推荐)
1. FC06 0x40C0=档位, 0x4040=平均次数, 0x4340=速度 (全段共用)
2. FC10 0x4400 写频点表 (N个16位M/E码, 高频→低频或任意序)
3. FC06 0x43C0 = N (点数)
4. FC05 线圈 0x00C0 = 0xFF00 (启动扫频)
5. 轮询: FC03 0x3E41 读已完成点数 (边跑边读), 或 FC03 0x3E40 等于2 (全部完成)
6. 批量读: FC03 0x3400+idx*4 (RE) / 0x3500+idx*4 (IM) / 0x3600+idx (VZM) / 0x3640+idx (实际频率码)
7. (可选) FC05 线圈 0x00C0 = 0 (中止)

> 旧法(上位机逐点重复8.2)仍可用, 不受影响。固件扫频省去逐点来回开销; 总测量时间不变(低频转换时间是物理硬底)。

### 8.4 采样电阻校准
1. 电桥量某档实际阻值 → FC06 0x40D0/D1/D2 = 实测mΩ
2. FC05 线圈 0x0010 = 0xFF00 (存Flash)
3. 掉电重启自动加载

### 8.5 OTA在线升级
见 `bootloader/README.md` (0x7000~0x7100 协议 + 0x0020 触发)。

## 9. 异常响应
标准 Modbus 异常: FC|0x80 + 异常码 (0x01=非法功能 0x02=非法地址)。
